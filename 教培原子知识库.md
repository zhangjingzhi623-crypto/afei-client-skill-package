# 教培原子知识库
> 定义：原子知识 = 最小可复用单元（可直接拼装为脚本、话术、活动、SOP）

## A. 认知原子（判断层）

1. **招生本质原子**：不是卖课，是影响家长决策心智。
2. **体验课原子**：家长买的不是低价，是“可见结果”。
3. **续费原子**：续费从开课第一天开始。
4. **转介绍原子**：先情感价值，再利益价值。
5. **经营原子**：预收款不是利润，增长要看真实课消与留存。

## B. 引流原子（入口层）

1. **低门槛入口**：1元/9.9/29.9。
2. **礼品原子**：刚需 > 贵重（红领巾/书皮/驱蚊/文具）。
3. **围观原子**：先让孩子停留，再让家长沟通。
4. **转发原子**：现场引导朋友圈/本地群传播。
5. **异业互利原子**：你给流量，他给场景或分润。

## C. 成交原子（转化层）

1. **真诚开场原子**：降低家长防御。
2. **痛点追问原子**：先问目标与顾虑再谈课程。
3. **成果具象原子**：每节课结束后孩子能得到什么。
4. **小决策原子**：大单拆小单（年包→月卡/短包）。
5. **补差价原子**：先入门，后升级（降低当下决策压力）。

## D. 续费原子（留存层）

1. **奖学金原子**：出勤即累积价值。
2. **阶段总结原子**：中途就做学习成果对比。
3. **三次机会原子**：一期内至少三次下单触发。
4. **氛围原子**：海报/倒计时/榜单/喜报同步。
5. **稀缺原子**：课位/礼品/时限形成明确窗口。

## E. 转介绍原子（裂变层）

1. **尊重原子**：先感谢老家长“帮忙背书”。
2. **承接原子**：专属群+专属服务+快速响应。
3. **可兑现原子**：让推荐动作有即时反馈。
4. **同圈层原子**：学校/小区身份标签增强信任。
5. **持续原子**：转介绍是机制，不是一次活动。

## F. 内容原子（传播层）

1. **钩子原子**：开头直击“痛点/利益/反常识”。
2. **结构原子**：问题—拆解—工具—行动。
3. **证明原子**：案例/反馈/数字优先。
4. **CTA原子**：给明确下一步（领工具/进群/到店）。
5. **复用原子**：同主题做系列，不做单条孤品。

## G. 组织原子（管理层）

1. **岗位边界原子**：市场/教学/教务职责明确。
2. **增量负责原子**：核心岗位对增长结果负责。
3. **薪酬绑定原子**：收入与出勤/续费/增量挂钩。
4. **流程标准原子**：把“人治”变“流程治”。
5. **复盘原子**：周复盘、月校准、季调整。

## H. 财务原子（生存层）

1. **现金流原子**：先活着，再扩张。
2. **成本原子**：固定成本不过载。
3. **结构原子**：短周期产品提升周转。
4. **风险原子**：避免靠预收堆虚胖增长。
5. **健康原子**：利润率和留存率同步看。
